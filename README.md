# 🍔 Online Food Ordering System (SQL Project)

## 📌 Overview

This project is a database-driven **Online Food Ordering System** built using **SQL in MySQL Workbench**. It focuses on storing, managing, and analyzing customer and order-related data efficiently.

The system allows capturing customer information and maintaining records for future use, enabling better data tracking and insights.

---

## 🎯 Objectives

* Store customer details and order information in a structured database
* Practice and demonstrate core SQL concepts
* Enable efficient querying and data retrieval
* Simulate a real-world food ordering backend system

---

## 🛠️ Technologies Used

* **Database:** MySQL
* **Tool:** MySQL Workbench
* **Language:** SQL

---

## 🗂️ Database Structure

The project includes multiple tables such as:

* **Customers** – Stores customer details
* **Orders** – Stores order information
* **Menu** – Contains food items and pricing
* **Order Details** – Tracks items within each order

---

## 🔑 Key Features

* Data storage for customers and orders
* Relational database design using primary and foreign keys
* Efficient data retrieval using advanced SQL queries

---

## 📊 SQL Concepts Applied

This project demonstrates strong use of:

* **Joins** (INNER JOIN, LEFT JOIN)
* **Clauses** (WHERE, GROUP BY, ORDER BY, HAVING)
* **Subqueries**
* **Aggregate Functions** (SUM, COUNT, AVG)

---

## 📈 Example Queries

Some of the analysis performed in this project includes:

* Retrieving customer order history
* Finding top-selling food items
* Calculating total revenue
* Identifying frequent customers

---

## 🚀 How to Run the Project

1. Open **MySQL Workbench**
2. Create a new database (e.g., `food_ordering_system`)
3. Import or run the provided SQL script file
4. Execute queries to view and analyze data
